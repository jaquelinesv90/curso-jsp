O sistema foi desenvolvido com o intuito de ser uma ferramenta
particular de gerenciamento de chamados.
Com esse sistema será possível criar chamados, editar e excluir.
A tela inicial apresenta uma listagem de ordem decrescente a criação.

Tecnologias utilizadas:
- Java (Servlet)
- Jstl
- Javascript
- Html, CSS
- JDBC
- Padrão Singleton
- o arquivo de conexão(SingleConnection) está configurado para ser utilizado
em PostgreSQL.
